# Combinators and Algorithms

**By Conor Shakory**  
*© 2026*

An online e-book exploring the elegant world of combinators and their
applications in algorithm design.

## View Online

Build and serve locally:

```bash
make serve
```

Then visit http://localhost:8000

## Build

See [BUILDING.md](BUILDING.md) for detailed instructions.

```bash
# HTML
make html

# PDF
make pdf
```
